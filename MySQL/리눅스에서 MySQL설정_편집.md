# 리눅스에서 MySQL설정_편집



VMware 실행 - root 로그인 - etc 폴더에 있는 my.cnf 편집

명령어

```
# cd /etc - 이동
# vi my.cnf - my.cnf를 vi로 실행한다.

```

실습에서 vi 편집기 사용된 명령어

vi 실행 후 처음 화면은 읽기 모드에서 시작이다.

| 명령어    | 기능                                  |
| --------- | ------------------------------------- |
| i         | 현재 커서 위치에서 insert 할 수 있다. |
| `esc`버튼 | 편집 모드에서 읽기 모드로 이동        |
| :w        | 추가한 내용 저장                      |
| :q        | vi 종료                               |
