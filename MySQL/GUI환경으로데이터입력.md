# MySQL데이터 입력(GUI)

MySQL에서 INSERT INTO 를 사용하지 않고 데이터를 입력할 수 있는 편안한 방법을 제공합니다.

아래 화살표에 보이는 버튼을 누르면 각 행에 데이터를 직접 입력할 수 있습니다.

입력 후에는 Apply 버튼을 누르고 데이터 타입과 크기에 맞게 입력했다면 오류 없이 데이터가 저장됩니다.
