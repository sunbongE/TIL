# 시스템 정보 함수



### USER()

> 현재 사용자를 반환



### DATABASE()

> 선택된 데이터베이스 반환



### FOUND_ROWS()

>  바로 앞에서 실행한 SELECT 문에서 조회된 행의 개수 반환



### ROW_COUNT()

> 바로 앞에서 실행한 INSERT, UPDATE, DELETE문에서 작업된 행의 개수를 구한다.
>
> CREATE, DROP : 0 반환
>
> SELECT : -1 반환



### VERSION()

> MySQL버전 반환



### SLEEP(S)

> 쿼리의 실행을 잠시 멈춤



```mysql
SELECT 'Hi';
SELECT SLEEP(2); -- 2초 멈추고 아래 문장을 수행
SELECT '2초 지남';
```
