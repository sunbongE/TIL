# 데이터 타입

# 핵심 정리

> 각 데이터 크기에 맞게 데이터 타입을 할당해야 성능좋고 메모리 낭비가 없는 프로그래밍을 할 수 있다.

```java
byte a = 127;
byte b = 1;
// 이라고 할 때 두 변수가 할당하는 크기는 같다. 왜냐면 byte타입이 1바이트 메모리를 차지하기 때문이다.
```

## 실수형

> 실수형을 사용 할 떄는 정확성을 위해서 double타입을 사용하는 것을 권장한다.

## 문자형

> 문자 하나에 2바이트 크기를 차지한다.

예를 들면

```java
char a = 'A';  // 2 바이트 
char b = "AB"; // 4 바이트
```
