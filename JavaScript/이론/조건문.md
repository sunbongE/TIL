# 조건문

**if**

조건 표현식의 결과 값을 불리언 타입으로 변환 후 참 거짓 판단.

---

**switch**

조건 표현식의 결과값이 어느 값에 해당하는지 판별

주로 특정 변수의 값에 따라 조건을 분기할 때 활용

조건이 많을 때 if보다 가독성이 나을 수 있다.

---

**if, else if, else**

조건은 소괄호 안에 작성

실행할 코드는 중괄호 안에 작성

블록 스코프 생성

----

switch statement

표현식의 결과값을 이용한 조건문

표현식의 결과값과 case문의 오른쪽 값을 비교

break문을 만나거나 default 문을 실행할 때까지 다음 조건문 실행
