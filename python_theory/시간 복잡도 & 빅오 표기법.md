# 시간 복잡도 & 빅오 표기법



## 시간 복잡도(알고리즘 수행시간)

>개개인의 환경에 따라 같은 알고리즘이라도 시간이다르다.(객관적 기준 필요)
>
>그래서 만듬

측정법

1. 기본연산이 몇번인지 (기본연산의 총 횟수 == 알고리즘의 소요 시간)
2. 성능을 측정할 때 입력을 통일시킴, (입력n에 따른)



##  빅오(Big-O) 표기법

입력 n이 무한대로 커진다고 가정하고 시간복잡도를 간단히 표시한 것

최고차항만 남기고 계수와 상수 제거하여 나타낸다.

증가율에 초점을 맞춰야한다.







**(가우스의 합 공식) = (n*(n+1)) // 2**
